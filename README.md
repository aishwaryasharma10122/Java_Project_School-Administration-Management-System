# School-Administration-Management-System
School Administration Management System (OOP Project)

##  Title
Design and Implementation of a School Administration Management System using Inheritance and Polymorphism in Object-Oriented Programming

---

##  Description
This project is a simple Java-based application developed to demonstrate key Object-Oriented Programming (OOP) concepts such as inheritance and polymorphism.

The system models a school environment by defining a base class `Person` and extending it into two specialized classes: `Teacher` and `Student`. Each class has its own properties and behavior.

The project also demonstrates runtime polymorphism using a `Person` reference, where different objects are handled dynamically at runtime.

---

## 🎯 Objectives
- Understand inheritance and class hierarchy  
- Implement method overriding  
- Demonstrate runtime polymorphism  
- Improve code reusability and structure  

---

## 🛠️ Technologies Used
- Java (JDK)
- VS Code / Eclipse (IDE)

---

## 🏗️ Project Structure
- `Person` (Base Class)
- `Teacher` (Derived Class)
- `Student` (Derived Class)
- `Main` (Driver Class)

---

## ⚙️ Features
- Input using Scanner class  
- Use of inheritance for code reuse  
- Method overriding for role-specific behavior  
- Runtime polymorphism using array of Person  
- Dynamic method dispatch
